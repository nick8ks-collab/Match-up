# MatchUp — Web (local-first)

## Dev (local)
```
npm install
npm run dev
```

## Build
```
npm run build && npm run preview
```

## Deploy (click-only)
- Push this folder to a new GitHub repo.
- Go to vercel.com → Add New Project → Import your GitHub repo → Framework: Vite → Build: `npm run build` → Output: `dist` → Deploy.
- Public RSVP link is simply your site URL with `?mode=rsvp`.
